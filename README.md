<summary>Q001. 1~n까지의 합</summary><br/>

한 정수 n을 입력 받아서 '1~n'까지의 합을 구하여 출력하시오. (수행 시간 출력)<br/>
단, 입력 방법은 구현하지 않아도 되며, 입력 값을 변수를 선언하여 사용해도 됩니다.<br/>
<br/>
1. Recursion<br/>
2. Repetition<br/>
<br/>
입력: 정수 n(2 ≤ n ≤ 10,000)이 첫 번째 줄에 입력됩니다.<br/>
출력: 1 ~ n까지의 합을 정수로 출력하시오.<br/>
<br/>
입력: 10<br/>
결과(Recursion): 55<br/>
결과(Repetition): 55<br/>
